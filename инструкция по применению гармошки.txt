Настройка:
В конфиге вставьте в поле URL URL битвы в скобочка, как сейчас стоит в самом конфиге.
ПКМ по батнику -> строка 8. Вместо слов ЛОГИН и ПАРОЛЬ вставить свои данные. Должно получиться так: -d "login=MyTheBestLogin&password=MyTheBestPassword&facebookAdId="


Для смены оружия смените weaponQuality с 0 на нужное значение. 0-это без оружия, 5 - самое мощное. 26 строка после метки :start
Для смены режима боя с берсерка измените параметр value с Berserk на undefined
Примеры data:
data = "weaponQuality=0&battleRoundId=23528&side=side&value=Berserk" атака берсерком без оружия на раунд 23528
data = "weaponQuality=3&battleRoundId=23537&side=side&value=undefined" Обычная атака оружием Q3 на раунд 23537


Файлы:
Config - сюда вставляй ЮРЛ
Otvet - файл-ответ с сервера. В нём обычно пишется подробная ошибка и что не так.
result - строка из файл-ответа. Используется для отображения логики.
Для того, чтобы они не появлялись уберите в батнике все значения ">result.txt", а в config уберите строку output = "otvet.txt"
